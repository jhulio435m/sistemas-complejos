# Dinámica de Sistemas

Este módulo introduce a la **Dinámica de Sistemas**, una metodología y técnica de simulación por ordenador para enmarcar, comprender y discutir problemas complejos y de larga duración.

## Enfoque Metodológico

La Dinámica de Sistemas fue creada por **Jay W. Forrester** en el MIT en la década de 1950. Se centra en las estructuras de retroalimentación interna de los sistemas para comprender cómo el comportamiento de un sistema es generado a lo largo del tiempo.

*   **Modelado:** Se utilizan diagramas de bucles causales y diagramas de Forrester (flujos y niveles) para visualizar la estructura del sistema.
*   **Simulación:** Se emplean herramientas como Vensim o Stella para simular el comportamiento del sistema a lo largo del tiempo y probar políticas alternativas.
*   **Arquetipos:** Se estudian patrones de comportamiento recurrentes en los sistemas, como "Límites del Crecimiento" o "Éxito para quien tiene Éxito".

## Aplicaciones



La Dinámica de Sistemas se aplica en una amplia variedad de campos:



*   **Estrategia Corporativa y Planificación:** Para entender las consecuencias a largo plazo de las decisiones.

*   **Políticas Públicas:** Para analizar el impacto de las políticas en áreas como la salud, la energía o el medio ambiente.

*   **Análisis de la Cadena de Suministro:** Para modelar y optimizar los flujos de materiales e información.
