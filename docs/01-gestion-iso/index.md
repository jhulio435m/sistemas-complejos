# Sistemas Integrados de Gestión (HSEQ)

Repositorio abierto de documentación técnica y estándares internacionales para la ingeniería de sistemas de gestión.

## Propósito
Centralizar recursos para el estudio, implementación y auditoría de sistemas basados en la Estructura de Alto Nivel (HLS). Este módulo se enfoca en la interoperabilidad técnica entre Calidad, Seguridad, Medio Ambiente y TI.

## Alcance Técnico
La documentación abarca la integración de los siguientes pilares normativos:

1.  **ISO 9001 (Calidad):** Optimización de procesos y enfoque al cliente.
2.  **ISO 14001 (Medio Ambiente):** Gestión de impactos y sostenibilidad.
3.  **ISO 45001 (Seguridad):** Prevención de riesgos laborales.
4.  **ISO 27001 (Seguridad de la Información):** Protección de activos digitales y gestión de riesgos TI.
